# @nostrwatch/demo-kit-with-idb

This is a demo of `@nostrwatch/kit` with the `@nostrwatch/kit-adapter-idb` adapter.

## Usage 
<details>

<summary>pnpm</summary>

```
pnpm run install 
pnpm run dev
```

</details>

<details>

<summary>yarn</summary>

```
yarn install 
yarn dev
```

</details>


<details>

<summary>npm</summary>

```
npm run install 
npm run dev
```

</details>
